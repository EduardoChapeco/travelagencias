# Inventário de Domínios do Sistema

## 1. Domínios Principais
- **Viagens (trips):** Cadastro de roteiros, localizadores de voo, acomodação.
- **CRM (crm):** Leads, estágios do funil, propostas.
- **Financeiro (financial):** Caixa, conciliações, invoices.
- **Vouchers (vouchers):** Estúdio de criação, importação de PDFs.
