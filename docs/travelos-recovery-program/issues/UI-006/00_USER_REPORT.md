# RELATO DO USUÁRIO - INCIDENTE DE INBOX COM QUEBRA GLASS E TELAS BRANCAS

## 1. Descrição do Relato (Hipótese)
O usuário relata que a caixa de entrada (Inbox `/agency/:slug/inbox`) perdeu sua identidade glass/dark do workspace, apresentando superfícies totalmente brancas opacas, textos invisíveis ou de baixo contraste, e a ausência da barra lateral (sidebar) estruturada.

## 2. Comportamento Esperado
* O módulo Inbox deve herdar os padrões de visual translúcido (glass) do workspace.
* Sem painéis brancos puros estáticos que ofusquem e fujam da identidade visual dark do painel.
* Todos os textos das conversas e mensagens devem ser plenamente visíveis e legíveis (sem texto branco sobre fundo claro).
