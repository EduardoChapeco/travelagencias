# RELATO DO USUÁRIO - COMPRESSÃO DE TEXTO NA HOME PÚBLICA

## 1. Descrição do Relato (Hipótese)
O usuário relata que a landing page pública (Home `/`) está gravemente quebrada, apresentando os textos comprimidos em uma coluna central extremamente estreita, forçando a quebra de uma palavra por linha.

## 2. Comportamento Esperado
* Títulos (`h1`) e parágrafos explicativos (`p`) devem cobrir sua largura normal de leitura (`max-w-4xl` e `max-w-3xl`) de forma centralizada e responsiva.
* Sem contaminação por regras de estilo ou largura restritiva da área administrativa do painel.
