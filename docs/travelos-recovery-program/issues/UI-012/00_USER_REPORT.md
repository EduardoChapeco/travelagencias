# RELATO DO USUÁRIO - CRM KANBAN GAPS E CONTRASTE

## 1. Descrição do Relato (Hipótese)
O usuário relata que o quadro Kanban de CRM (`/agency/:slug/crm`) exibe colunas com gaps (espaçamentos) inconsistentes, larguras de coluna inadequadas que espremem os dados do lead, e baixo contraste cromático nos badges de status e tags de valor.

## 2. Comportamento Esperado
* Gaps e margens de layout consistentes em todo o painel de CRM.
* Largura de colunas adequada no Kanban, permitindo a leitura legível das informações do lead (nome, valor, data do último contato).
* Badges de prioridade, status e valor com contraste adequado (atendendo à taxa 4.5:1 de legibilidade).
