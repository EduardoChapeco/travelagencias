# RELATO DO USUÁRIO - INCIDENTE DE HEADER E TOOLBAR

## 1. Descrição do Relato (Hipótese)
O usuário relata diversos problemas no cabeçalho (Header) e barra de filtros (Toolbar): pills de filtro cortadas, botões e controles sem ícone ou texto de suporte, componentes sobrepostos, e a toolbar central desalinhada ou com altura divergente das islands laterais.

## 2. Comportamento Esperado
* Alinhamento horizontal limpo e centralizado no header do AppShell.
* Pills de filtro e controles operacionais totalmente visíveis, legíveis e com contraste adequado.
* Sem sobreposição com o logotipo da agência (à esquerda) ou o widget de hora/data (à direita).
* Altura coerente e uniforme de todos os elementos que compõem o cabeçalho.
