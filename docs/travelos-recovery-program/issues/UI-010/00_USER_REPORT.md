# RELATO DO USUÁRIO - CARD BRANCO E PAGINAÇÃO EM CONTRATOS

## 1. Descrição do Relato (Hipótese)
O usuário relata que o módulo de Contratos (`/agency/:slug/contracts`) exibe um card branco incompleto, botões vazios e layout estático, com baixo contraste e paginação comprometida.

## 2. Comportamento Esperado
* Interface do módulo de contratos seguindo o tema glass/dark do workspace, sem cards brancos de alto contraste ofuscantes.
* Botões com ícones legíveis e ações associadas visíveis.
* Paginação operável e com feedback claro.
