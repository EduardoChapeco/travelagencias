# RELATO DO USUÁRIO - DOCK LATERAL RIGIDO

## 1. Descrição do Relato (Hipótese)
O usuário relata que o dock vertical flutuante compacto foi substituído por uma barra lateral permanente larga que reserva uma coluna inteira no grid do workspace e exibe um painel contextual rígido com o título "Menu de Contexto".

## 2. Comportamento Esperado
* O dock nos módulos internos deve aparecer verticalmente no lado esquerdo.
* Deve manter o formato de pill/card vertical flutuante compacto e cantos arredondados, sem virar uma sidebar tradicional larga.
* Não deve reservar uma coluna enorme de grid no layout.
* Ao clicar em "Voltar" dentro do próprio dock, os itens principais devem substituir os itens contextuais na mesma visualização de dock sem forçar a navegação de página.
