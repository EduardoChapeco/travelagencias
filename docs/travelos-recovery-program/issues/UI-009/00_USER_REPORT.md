# RELATO DO USUÁRIO - TABELA BRANCA EM VIAGENS

## 1. Descrição do Relato (Hipótese)
O usuário relata que a listagem de Viagens (`/agency/:slug/trips/`) apresenta uma tabela branca opaca gigante sobre o plano de fundo, com textos e ícones invisíveis ou de baixíssimo contraste, e botões de paginação sem ícone/legenda inteligível.

## 2. Comportamento Esperado
* A tabela de listagem de viagens deve possuir estilo glassmorphic transparente (conforme design system do OS), integrando-se suavemente sobre o wallpaper.
* Legibilidade completa das colunas, status e localizadores.
* Ícones de paginação (anterior/próxima/primeira/última) perfeitamente visíveis e acionáveis.
