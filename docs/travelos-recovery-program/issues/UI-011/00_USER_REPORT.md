# RELATO DO USUÁRIO - TABELA VAZIA E INCOMPLETA EM COTAÇÕES

## 1. Descrição do Relato (Hipótese)
O usuário relata que a Central de Cotações (`/agency/:slug/quotes`) exibe uma tabela praticamente vazia ou invisível, com linhas brancas sem conteúdo e paginação desalinhada ou inoperante.

## 2. Comportamento Esperado
* Listagem transparente e translúcida das cotações ativas da agência.
* Texto e status de cada orçamento (planejado, enviado, aprovado) perfeitamente visíveis.
* Paginação visível e com ícones funcionais.
