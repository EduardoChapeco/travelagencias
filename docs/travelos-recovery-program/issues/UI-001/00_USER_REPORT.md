# RELATO DO USUÁRIO - INCIDENTE DE BLUR GLOBAL

## 1. Descrição do Relato (Hipótese)
O usuário relata que o plano de fundo (wallpaper) do sistema está sendo desfocado (blur) de forma global e incondicional no workspace inteiro. Isso inviabiliza o controle fino do desfoque por meio das preferências de tema do usuário.

## 2. Comportamento Esperado
* Wallpaper nítido (sem blur) por padrão.
* Intensidade de desfoque e escurecimento (dim) do wallpaper controlados exclusivamente pelo painel de personalização do usuário.
* O desfoque "frost glass" deve residir apenas nas superfícies dos cards/painéis operacionais, sem espalhamento global.
* Nenhuma regra global no workspace deve aplicar blur automático a toda a tela.
