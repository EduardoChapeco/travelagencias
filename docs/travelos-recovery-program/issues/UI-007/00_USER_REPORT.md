# RELATO DO USUÁRIO - RADII DE CÁPSULAS NO KANBAN DE EMBARQUES

## 1. Descrição do Relato (Hipótese)
O usuário relata que as colunas do quadro Kanban de embarques estão com bordas excessivamente arredondadas (formato de cápsulas gigantes), quebrando a harmonia visual com os cards internos e o restante da interface.

## 2. Comportamento Esperado
* Colunas do Kanban com bordas quadrangulares ou com border-radius leve (`rounded-2xl` ou `radius-card` de 28px).
* Formatos arredondados totais (`rounded-full`) reservados exclusivamente para botões e inputs de pílulas, conforme definido no DESIGN.md.
